# Web Pool D06 - JavaScript modules

## Description

Ce depot contient des exercices JavaScript organises en modules `.mjs`, avec des fichiers de test associes.

## Objectif

L'objectif est de pratiquer les fonctions JavaScript, les exports de modules et la manipulation de valeurs simples.

## Stack / technologies

- JavaScript
- Modules ES (`.mjs`)
- Node.js

## Structure

```text
.
└── day06/
    ├── task01.mjs
    ├── task01_tester.mjs
    ├── task02.mjs
    ├── task02_tester.mjs
    ├── task03.mjs
    ├── task03_tester.mjs
    ├── task04.mjs
    ├── task05.mjs
    └── task05_tester.mjs
```

## Fonctionnalites principales

- Fonctions exportees.
- Exercices de logique JavaScript.
- Testeurs fournis pour verifier certains exercices.

## Lancement

Executer un testeur avec Node.js :

```bash
node day06/task01_tester.mjs
```

## Remarques

Les exercices sont independants et suivent une progression par `task`.
